## 📌 About

This repository provides optimization tweaks for Palworld to increase FPS, reduce stuttering, and improve overall performance, especially on mid‑range and low‑end hardware. All changes are safe and do not interfere with anti‑cheat systems.

What it does:  
✅ Optimized Engine.ini settings for Unreal Engine 5  
✅ Reduces input lag and frame drops  
✅ Adjusts streaming and shadow quality  
✅ Easy backup and restore

---

## ⚙️ How to Use

1. [Download](Link)
2. Run optimize.bat as administrator.
3. Choose:
   - 1 – Apply performance tweaks  
   - 2 – Restore original settings  
   - 3 – Exit

The script automatically backs up your original Engine.ini before making changes.

> 📁 Manual installation: copy config/Engine.ini to  
> %LOCALAPPDATA%\Palworld\Saved\Config\WindowsClient\

## 🛡️ Important

- This tool does not modify game executables – no anti‑cheat issues.
- All tweaks are reversible via the restore option.
